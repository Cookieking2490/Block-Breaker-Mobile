Brick Breaker Mobile Game

A 2D Unity brick breaker game developed for KH6004CMD Mobile App Development coursework.

Features:
- User login
- Brick breaking gameplay
- Score system
- Backend API integration

Technologies:
- Unity
- C#
- REST API
- Firebase

Steps to run:
1. Clone repository
2. Open project in Unity Hub
3. Open MainScene
4. Press Play
