# 🎮 Brick Breaker Mobile Game

A 2D mobile Brick Breaker game developed using Unity for the **KH6004CMD Mobile App Development coursework**.
The game features user authentication, multiple levels, score tracking, and cloud data storage using Firebase.

---

## 🚀 Features

* 🔐 User login and registration (Firebase Authentication)
* 🧱 Classic brick-breaking gameplay
* 🎯 Multiple playable levels
* 📊 Score system with high score saving (Firebase Firestore)
* 🔊 Sound effects for paddle and brick collisions
* ✨ Particle effects for brick destruction
* 🎨 Improved visuals using custom sprites and background
* 📱 Mobile-optimized UI and controls

---

## 🛠️ Technologies Used

* Unity Engine
* C#
* Firebase Authentication
* Firebase Firestore
* TextMeshPro (UI system)

## ▶️ How to Run

1. Clone the repository
2. Open the project using **Unity Hub**
3. Open the main scene (e.g., `MainScene`)
4. Press **Play** in Unity Editor

---

## 🎮 Gameplay Overview

* Control the paddle to bounce the ball
* Break all bricks to complete the level
* Earn points and try to beat your high score
* Progress through multiple levels
